1512. Number of Good Pairs
Easy
Array

Количество хороших пар

Хорошая пара это когда две цифры совпадают nums[i] == nums[j] и i < j мы реализуем так $j = $i + 1
получается нужно вывести результат сколько таких пар

выполнение
создаем переменную сумма с начальным значением 0

создаем цикл для i элементов массива
создаем внутренний цикл для  j элементов массива то есть берем все значения равные текущее значение + 1
то есть все значения следующие прямо за текущим значением
и получается что

[1,2,3,1,1,3] входящий массив
общая работа цикла с трассировкой
1 сравниваем с 2,3,1,1,3 если что то совпало то плюс 1 к сумме
2 == 3,1,1,3
3 == 1,1,3
1 == 1,3
1 == 3

Input: nums = [1,2,3,1,1,3]
Output: 4
Explanation: There are 4 good pairs (0,3), (0,4), (3,4), (2,5) 0-indexed.
