1748. Sum of Unique Elements
Easy
Array Hash Table

Описание

Вам дан целочисленный массив nums. Уникальные элементы массива - это элементы,
которые появляются в массиве ровно один раз.

Вернуть сумму всех уникальных элементов числа.

Выполнение

* Создаем переменную результат sum = 0 в конце ее вернем
* Копируем входящий массив $a = $nums;
* Пробегаемся по копии циклом
* Создаем счетчик который считает сколько раз встречается цифра
* Пробегаемся по входящему массиву внутри первого цикла чтобы сравнить
    если одна из входящих цифр равна цифре внутри копии то добавляем один к счетчику

    Трассировка циклов

    input arr [1,2,3,2]
    copy arr [1,2,3,2]

    step 1
    1 == 1 count = 1
    1 == 2 count = 1
    1 == 3 count = 1
    1 == 2 count = 1

    step 2
    2 == 1 count = 0
    2 == 2 count = 1
    2 == 3 count = 1
    2 == 2 count = 2

    step 3
    3 == 1 count = 0
    3 == 2 count = 0
    3 == 3 count = 1
    3 == 2 count = 1

    step 4
    2 == 1 count = 0
    2 == 2 count = 1
    2 == 3 count = 0
    2 == 2 count = 2

* Выходим из второго цикла имеем готовый счетчик который постичал сколько раз цифоа повторилась
создаем условие если цифра появилась меньше или равно 1 то к переменной которую мы создали в начали
конкатенируем цифры копии входящего массива то есть находим сумму всех уникальных цифр

* возвращаем сумму
