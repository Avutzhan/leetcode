1313. Decompress Run-Length Encoded List
Easy
Array

Распаковать закодированный список длин серий,
то есть список цифр закодированных в массиве,
то есть это шифр [1,2,3,4] количество всегда четное
который означает
1 - это сколько раз вывести 2
3 - это сколько раз вывести 4

выполнение
создаем переменную результат с пустым массивом

создаем цикл и пробегаемся по всем вторым эелментам входящего массива
то есть по значениям которые мы собираемся выводить
так как входящий массив всегда четный мы не паримся и делаем шаг через 2

создаем цикл внутри цикла который будет пробегаться по первым элементам
это наша частота добавления значения то есть сколько раз мы выводим

и привязываем эту цифру к условию цикла то есть второй цикл длится ровно столько
на сколько равна эта цифра шаг обычный j++
и внутри второго цикла тупо пушим в массив те цифры значния по которым мы пробегаемся
в первом цикле

и возвращаем результат

Input: nums = [1,2,3,4]
Output: [2,4,4,4]
Explanation: The first pair [1,2] means we have freq = 1 and val = 2 so we generate the array [2].
The second pair [3,4] means we have freq = 3 and val = 4 so we generate [4,4,4].
At the end the concatenation [2] + [4,4,4] is [2,4,4,4].
