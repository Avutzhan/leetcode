1470 Shuffle the Array
Easy
Array

Тасовать Смешать массив

Дан массив цифр и количество пар в массиве в виде цифры.
Нужно смешать массив разделив его на две равные части и с каждой части
выбираем соответствующую цифру и добавляем в новый массив
сперва выберем первый элемент первой половины потом первый элемент второй половины

выполнение
создаем переменную результат с пустым массивом

создаем цикл который пройдется по входящему массиву
столько раз сколько пар в массиве $n второй аргумент принимающий наша функция
в массив добавляем поочередно внутри цикла сперва текущий элемент
а потом за ним элемент следующий плюс шаг то есть количество пар в массиве
если отсчитать от текущего элемента количество пар то получим его пару в другой половине массива

выходим из цикла и возвращаем массив результат

Input: nums = [2,5,1,3,4,7], n = 3
Output: [2,3,5,4,1,7]
Explanation: Since x1=2, x2=5, x3=1, y1=3, y2=4, y3=7 then the answer is [2,3,5,4,1,7].
